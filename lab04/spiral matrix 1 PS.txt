Given an n x n matrix, return all elements of the matrix in spiral order.

Example 1:



Input:

matrix = [[1,2,3],[4,5,6],[7,8,9]]

Output:

[1,2,3,6,9,8,7,4,5]

Constraints:

n == matrix.length

n == matrix[i].length

1 <= n <= 10

-100 <= matrix[i][j] <= 100
For example:

Input	Result
3
1 2 3
4 5 6
7 8 9
1 2 3 6 9 8 7 4 5
